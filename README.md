# algprog-lista-03-java

Tarefa na qual tive que criar 6 algoritmos em Java:

- O primeiro lê 3 números digitados e define o maior, o menor e a média dentre eles;

- O segundo simula uma máquina de venda de salgadinhos;

- O terceiro resolve uma equação de segundo grau;

- O quarto calcula o perímetro, a área ou o volume de um círculo a partir da escolha do usuário;

- O quinto faz operação de adição, substração, multiplicação, divisão ou potenciação a partir da escolha do usuário;

- O último mostra qual o maior número dentre 2 digitados e depois sorteia um número aleatório entre 0 e 100, definindo se ele é par ou ímpar.
